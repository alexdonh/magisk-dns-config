# DNS Config

## Description

A simple module to configure DNS servers via terminal.
After installed via Magisk, enter the command in your favorite Terminal:

```
su
```

```
dnsconfig -s <DNS1> <DNS2>
```

To reset:

```
dnsconfig -u
```
